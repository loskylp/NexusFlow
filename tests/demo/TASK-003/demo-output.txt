--------------------------------------------------------------------------------

  https://nexusflow.staging.nxlabs.cc/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "username": "admin",
    "password": "admin"
  }' | python3 -m json.tool

Response: HTTP 200
Token obtained: 86ae47fa37444c64ee12d4e5df7746c19036de0d18333f5a95500ecd8b6be7b5

Step 2 -- Verify the token works (pre-logout):

$ curl -s -o /dev/null -w "%{http_code}" \
  https://nexusflow.staging.nxlabs.cc/api/tasks \
  -H "Authorization: Bearer 86ae47fa37444c64ee12d4e5df7746c19036de0d18333f5a95500ecd8b6be7b5"

Response: HTTP 200

Step 3 -- Logout:

$ curl -s -X POST \
  https://nexusflow.staging.nxlabs.cc/api/auth/logout \
  -H "Authorization: Bearer 86ae47fa37444c64ee12d4e5df7746c19036de0d18333f5a95500ecd8b6be7b5"

Response: HTTP 204 (No Content)

Step 4 -- Verify the token is invalid (post-logout):

$ curl -s \
  https://nexusflow.staging.nxlabs.cc/api/tasks \
  -H "Authorization: Bearer 86ae47fa37444c64ee12d4e5df7746c19036de0d18333f5a95500ecd8b6be7b5"

Response: HTTP 401
{"error":"unauthorized"}

RESULT: PASS
  - Session immediately revoked on logout.
  - Same token returns 401 after POST /api/auth/logout.
